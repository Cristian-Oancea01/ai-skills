# Global OpenCode Rules

## Default skills

Load these skills immediately at the start of every session, before doing any work:

- `coding-standards` — always load for any coding task

Load these skills only when needed:

- `agent-orchestration` — load for complex multi-step tasks that benefit from parallel agent coordination; skip for simple or single-file tasks
- `homeassistant-setup` — load when working with Home Assistant, QNAP NAS, or HA config files (skill lives in the `homeassistant-setup` project at `.opencode/skills/homeassistant-setup/SKILL.md`)
